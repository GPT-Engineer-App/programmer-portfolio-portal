# programmer-portfolio-portal

A personal website for a programmer. The website should showcase the programmer's portfolio, including projects, skills, and a blog section for sharing insights and updates. The design should be modern and professional, with a focus on usability and showcasing the programmer's expertise in software development.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/programmer-portfolio-portal.git
cd programmer-portfolio-portal
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
